# AI Recruitment Intelligence System (ARIS)


ARIS is an AI-powered platform that evaluates candidate profiles using:
- Resume extraction
- GitHub analysis
- Portfolio website analysis
- ATS scoring
- Job description matching

This is built using **FastAPI + React + SQLAlchemy + ML/NLP**.
